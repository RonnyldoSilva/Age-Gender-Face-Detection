# Age Gender Face Detection

See in action here: https://youtu.be/V1Z5dHkXyDs

### Opencv 3.4 is required.

### Run Code

#### C++
```
cmake .
make
./AgeGender <input_file>(Leave blank for webcam)
```

#### Python
```
python AgeGender.py --input <input_file>(Leave blank for webcam)
```
